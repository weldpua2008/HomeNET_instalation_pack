Команды в чате

Сообщения в чат

/clear
/cls
/c 		Очистить окно сообщений
/close Закрыть текущее окно
/me сообщение Хаб отправляет сообщение о пользователе от 3-го лица
/ratio
/r 		Статистика скачано/отдано за всё время использования клиента
/uptime
/ut 		Время, прошедшее от перезагрузки ОС
/stats Статистика по работе клиента и компьютера (не видна другим пользователям)
/pubstats Статистика по работе клиента и компьютера (видна другим пользователям)

Команды на хабе

/log Открывает лог-файл чата хаба
/log status Открывает лог-файл системных сообщений хаба
/find текст Искать текст в чате хаба
/join адрес хаба Подключиться к другому хабу по указанному адресу
/password пароль Пароль, требуемый при входе на хаб зарегистрированного пользователя
/pm ник сообщение Послать приватное сообщение или открыть окно сообщений
/getlist ник Скачать файл-лист
/favorite
/fav 		Добавить хаб в "избранное" (параметр /a - с автозагрузкой)
/removefavorite
/removefav 		Убрать хаб из "избранного"
/ts Включить/выключить временные метки перед сообщениями
/showjoins Включить/выключить показ входа и выхода пользователей из хаба
/favshowjoins Включить/выключить показ входа и выхода избранных пользователей
/userlist Показать/скрыть список пользователей
/switch Меняет местами список пользователей и чат хаба
/ignorelist Список заигноренных пользователей, которые есть на этом хабе
/ipclones Список пользователей, зашедших на хаб с одного IP-адреса (если IP-адреса известны клиенту)
/iphistory Ник
/iphistory IP 		История IP-адресов ника / ников на IP-адресе (если включено ведение истории IP в настройках)
На ADC-хабах под "ником" понимается CID, связанный с введённым ником

Команды в личных сообщениях

/grant Дать слот
/favorite
/fav		 Добавить в "Избранные"
/getlist
/gl		 Скачать файл-лист
/net Передать конфигурацию сетей (нужно для обмена по VPN, минуя сеть хаба)
/key Передать открытый ключ шифрования приватных сообщений. Ключ автоматически будет использован, если у собеседника установлен greylink 5.67 или выше. После обмена командами /key можно включать шифрование ЛС

Системные команды, управление клиентом

/lua выражение Вычисляет lua-выражение. Выражение может содержать вызовы функций
/luafile имя_файла Выполняет lua-скрипт из файла в каталоге scripts
/bot параметры Передаёт команды/параметры боту в ChatBot.dll. Реакция зависит от подключенного бота
/log system
/log uploads
/log downloads		 Открывает лог-файл
/connection Информация об ip-адресе и открытых портах TCP/UDP/TLS (не работает в окнах ЛС)
/hub [порт] Открыть чат-комнату (работает как хаб). Если порт не указан, выбирается случайный
/refresh Инициирует обновление шары
/away Включает/выключает статус "Ушёл"
/away сообщение Устанавливает сообщение "Ушёл" и включает статус "Ушёл"
/limit Включает/выключает ограничения скорости
/limit Исх Задаёт ограничение исходящей скорости (Кб/сек)
/limit Исх Вход Задаёт ограничение исходящей и входящей скорости (Кб/сек)
/slots число Установить количество слотов
/extraslots число Установить количество минислотов
/smallfilesize число Установить размер минислота
/search строка Начать поиск на хабах (файла или TTH)
/update
/up 		Проверить обновление на сайте клиента
/rebuild Оптимизация базы TTH (с предыдущими настройками, выставленными в меню Файл --> почистить базу TTH)
/rebuild scan Анализ базы данных - сколько места на диске занимают разные структуры
/shutdown Выключение компьютера (возможно, в следующих версиях исчезнет)
/help
/h		 Краткий список команд со ссылкой на этот документ
