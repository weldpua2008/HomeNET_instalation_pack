Управление с клавиатуры

Главное окно программы

Ctrl+1 Панель кнопок
Ctrl+2 Статус-панель
Ctrl+3 Список активных подключений
Ctrl+4 Панель медиаплеера
Ctrl+5 Быстрый поиск

Управление вкладками

Ctrl+W Закрыть активную вкладку
Ctrl+B Отменить закрытие последней вкладки
Alt+Left Предыдущая вкладка
Alt+Right Следующая вкладка
Alt+Shift+Left Перемещение текущей вкладки влево
Alt+Shift+Right Перемещение текущей вкладки вправо
Alt+цифра Переход на вкладку по номеру
Shift+Alt+цифра Обмен активной вкладки с выбранной
Alt+Up Циклическое вращение списка вкладок вправо
Alt+Down Циклическое вращение списка вкладок влево

Быстрый доступ к окнам

Ctrl+D Очередь закачек
Ctrl+Q Очередь отдачи
Ctrl+S Новое окно поиска
Ctrl+Shift+D Завершенные скачивания
Ctrl+Shift+U Завершенные отдачи
Ctrl+F Избранные хабы
Ctrl+P Интернет-хабы
Ctrl+U Избранные пользователи
Ctrl+G Настройка авто-поиска (ADL-поиск)
Ctrl+H Прогресс хеширования
Ctrl+I Глобальные настройки
Ctrl+N Блокнот
Ctrl+O Своя шара
Ctrl+Shift+H Новый запрос к базе данных скачиваний
Ctrl+Shift+T Новостной трекер

Кнопки в окне хаба

Ctrl+R Переподключение к текущему хабу
Ctrl+T Подключиться к хабу, на который дал направление текущий хаб при отключении
(если выключена настройка "автоматически следовать перенаправлениям")
Ctrl+Y Окно приватного чата с пользователем, который выделен в списке пользователей
Двойной клик в списке Настраивается в опциях. По умолчанию - список файлов, другие варианты: личное сообщение, ник в чат, выдать слот, добавить в друзья
Двойной клик ник в чате Настраивается в опциях. По умолчанию - найти в списке, другие варианты: личное сообщение, ник в чат, выдать слот, добавить в друзья
Ctrl + Двойной клик ник в чате Личное сообщение
Shift + Двойной клик ник в чате Загрузить список файлов

Кнопки в строке ввода чата

Alt+R Перекодировка раскладки клавиатуры (Пример <=> Ghbvth)
Alt+E Открыть панель смайлов
Alt+B Выделенный текст - в код [b]текст[/b]
Alt+I Выделенный текст - в код [i]текст[/i]
Alt+U Выделенный текст - в код [u]текст[/u]
Alt+S Выделенный текст - в код [s]текст[/s]
Alt+J Вставить картинку
Alt+L Ввести web-ссылку к выделенному тексту

Кнопки в окне файл-листа

Backspace Перейти в папку на уровень выше
Alt+влево Назад в истории навигации
Alt+вправо Вперёд в истории навигации
Двойной клик Скачать файл
Shift+контекстное меню В списке файлов - открыть меню проводника для выделенного файла, если он есть в шаре
Ctrl+контекстное меню Заменяет все пункты меню "скачать" на "скачать с приоритетом пауза"
Shift+контекстное меню Заменяет все пункты меню "скачать" на "скачать с наивысшим приоритетом" (в списке файлов не действует, если выбран один файл, т.к. задействован для меню проводника)

Кнопки в окне результатов поиска

Двойной клик Скачать файл
Ctrl+контекстное меню Заменяет все пункты меню "скачать" на "скачать с приоритетом пауза"
Shift+контекстное меню Заменяет все пункты меню "скачать" на "скачать с наивысшим приоритетом"

Кнопки в окне интернет-хабов

Alt+R Обновить список
Alt+C Настройки
Alt+I Курсор в поле "Фильтр"

Окно соединений

Двойной клик Настраивается. По умолчанию - личное сообщение, другие варианты: список файлов, поиск альтернатив, принудительное соединение

Дополнительные действия

Ctrl+E Обновить шару
Ctrl+L Открыть файл-лист с диска
Ctrl+M Ввести magnet-ссылку для поиска/скачивания
Alt+Q Подключиться к хабу (ввести адрес хаба)
Alt+D Закрыть окна отключенных хабов
Alt+P Закрыть окна приватных чатов отключенных пользователей
Ctrl+Shift+R Переподключить все отключенные хабы

Загрузка программы

Shift При удержании Shift в момент появления главного окна не происходит подключение к избранным хабам, отмеченным опцией "Авто-подключение"
